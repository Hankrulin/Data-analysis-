
# Mining_Big_Data

## Overview

This repository presents three course projects completed for **Mining Big Data**, covering key areas of data mining and machine learning: **classification, forecasting, and recommendation systems**.

Each project applies different analytical methods to solve a practical data problem and demonstrates the use of data preprocessing, modelling, evaluation, and interpretation techniques.

## Included Projects

### Project 1: Obesity Data Analysis
A classification project that investigates obesity levels using health and lifestyle-related features.  
The project includes data exploration, preprocessing, visualisation, and predictive modelling with machine learning algorithms.

### Project 2: Time Series Forecasting
A forecasting project that predicts household appliance energy consumption from time-indexed data.  
The project compares statistical and deep learning approaches, including ARIMA and LSTM, to model temporal patterns.

### Project 3: Grocery Recommendation System
A recommendation project based on grocery transaction data.  
The project combines frequent pattern mining, collaborative filtering, and hybrid recommendation techniques to generate product recommendations.

## Key Learning Outcomes

- Applied machine learning to classification tasks
- Built forecasting models for time series data
- Developed recommendation systems using multiple approaches
- Practised data preprocessing, feature engineering, and evaluation
- Gained experience working with different data types and problem settings
